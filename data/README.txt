Папка данных проекта.

complex_service_comments.json — тексты комментариев по умолчанию для услуг в форме «Комплексное ТКП».
  - Ключ: наименование услуги (как в БД: ДП, ДКП, Навигация и т.д.).
  - Значение: текст, подставляемый в поле «Комментарий» при выборе услуги.
  - При деплое файл копируется вместе с проектом; при сохранении в «Редактирование описания услуги» файл обновляется.
  - Если файла нет или услуги нет в файле, используются значения из БД или из кода (fallback).
