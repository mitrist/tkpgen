# Инструкция по работе с приложением «Генератор ТКП»

Приложение предназначено для формирования технико-коммерческих предложений (ТКП) в форматах DOCX и PDF: по одному из типовых шаблонов (по виду услуги) или в виде комплексного ТКП с таблицей позиций.

---

## Навигация

В левой панели доступны разделы:

| Раздел | Назначение |
|--------|------------|
| **Формирование ТКП** | Создание ТКП по выбранной услуге (типовой шаблон). |
| **Комплексное ТКП** | Создание ТКП с произвольным списком позиций и общей суммой. |
| **Перечень ТКП** | Просмотр и поиск ранее сформированных ТКП. |
| **Тарифы по регионам** | Настройка цен по услугам и регионам для расчёта стоимости. |

---

## 1. Формирование ТКП (типовое)

Пошаговый процесс: заполнение формы → проверка данных → формирование и скачивание файлов.

### Шаг 1. Заполнение формы

1. Откройте **Формирование ТКП** в меню.
2. Заполните поля:
   - **Дата ТКП** — дата документа.
   - **Услуга** — выберите вид услуги из списка (от него зависит шаблон и единица измерения).
   - **Регион** — выберите регион для расчёта стоимости по тарифу (для внешнего заказчика).
   - **Наименование клиента** — для внешнего заказчика.
   - **Наименование объекта и характеристика помещений** — при необходимости.
   - **Срок разработки** — при необходимости выберите из списка.
   - **Площадь (м²)** или **Количество (шт)** — в зависимости от услуги; используется для расчёта стоимости.
   - **Произвольный текст** — при необходимости.

**Внутренний заказчик:** отметьте «Является внутренним заказчиком?» — появятся поля «Внутренний клиент» и «Стоимость для внутреннего заказчика». Регион и площадь/количество для расчёта не требуются.

3. Нажмите **Сформировать ТКП**.

### Шаг 2. Подтверждение

На экране отображаются введённые данные и рассчитанная стоимость. Проверьте их.

- **Подтвердить и сформировать ТКП** — перейти к формированию и скачиванию.
- **Вернуться к форме** — изменить данные.

### Шаг 3. Скачивание

После формирования доступны кнопки:

- **Скачать PDF** — скачать ТКП в формате PDF.
- **Скачать DOCX** — скачать ТКП в формате Word.

Дополнительно: **Новое ТКП** (создать ещё одно) и **Перечень ТКП** (перейти к списку).

## 2. Комплексное ТКП

ТКП с произвольной таблицей позиций (наименование, единица измерения, количество, цена за единицу, стоимость). Итоговая сумма считается автоматически.

### Шаг 1. Заполнение формы

1. Откройте **Комплексное ТКП** в меню.
2. Заполните общие поля:
   - **Дата ТКП**
   - **Наименование клиента**
   - **Срок разработки** (Выпадающий список)
   - **Доп. текст 1** и **Доп. текст 2** (по желанию; если пусто — в документ не попадают).
   - **Доп. текст 1** вставляет текстовое значение ДО таблицы
   - **Доп. текст 2** вставляет текстовое поле ПОСЛЕ таблицы

3. Заполните таблицу позиций:
   - **Наименование услуги** — текст.
   - **Ед. изм.** — м² или шт.
   - **Количество** — число.
   - **Цена за ед.** — цена за единицу в рублях. Стоимость по строке считается автоматически.

4. При необходимости:
   - **Добавить строку** — новая строка в таблице.
   - Кнопка **×** в строке — удалить строку (должна остаться хотя бы одна).

5. Нажмите **Далее — подтверждение**.

### Шаг 2. Подтверждение

Проверьте дату, клиента, срок, доп. тексты и таблицу с итоговой суммой.

- **Подтвердить и сформировать ТКП** — сформировать документ и перейти к скачиванию.
- **Вернуться к форме** — изменить данные.

### Шаг 3. Скачивание

Аналогично типовому ТКП: можно скачать **PDF** и **DOCX**, перейти к **Новому ТКП** или в **Перечень ТКП**.

## 3. Перечень ТКП

Таблица всех сформированных ТКП с фильтрами.

### Колонки

- **Дата** — дата ТКП.
- **Время** — время формирования.
- **Номер документа** — уникальный номер.
- **Клиент** — наименование клиента.
- **Услуга** — вид услуги или «Комплексное ТКП».
- **Сумма** — итоговая стоимость (₽).

### Фильтры

Над таблицей расположены фильтры по столбцам:

- **Дата с** / **Дата по** — период по дате.
- **Номер** — поиск по части номера документа.
- **Клиент** — поиск по части названия клиента.
- **Услуга** — выбор услуги из списка (или «Все»).
- **Сумма от** / **Сумма до** — диапазон суммы.

Нажмите **Применить**, чтобы отфильтровать список. **Сбросить** — снять все фильтры.

Ссылка **Формирование ТКП** ведёт к форме создания нового ТКП.

---

## 4. Тарифы по регионам

Раздел для настройки цен за единицу по парам «Услуга — Регион». Эти тарифы используются при расчёте стоимости в разделе **Формирование ТКП** для внешнего заказчика.

### Добавление тарифа

1. В блоке **Добавить тариф** выберите **Услугу**.
2. Укажите регион:
   - выберите **Регион** из списка («— или выберите существующий —»), или
   - введите **Новый регион** — будет создан новый регион с указанным названием.
3. Введите **Цена за единицу (₽)**.
4. Нажмите **Добавить**.

### Удаление тарифа

В таблице текущих тарифов нажмите **Удалить** в нужной строке.
