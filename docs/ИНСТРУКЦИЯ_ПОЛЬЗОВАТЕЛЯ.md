# Инструкция по работе с приложением «Нацпро-webservice»

Приложение предназначено для формирования технико-коммерческих предложений (ТКП) и договоров в форматах DOCX и PDF: по одному из типовых шаблонов (по виду услуги) или в виде комплексного ТКП с таблицей позиций. Поддерживаются реестры ТКП и договоров, канбан-доска сделок и справочник контрагентов.

---

## Навигация

В левой панели (на мобильных — выпадающее меню) доступны разделы:

| Раздел | Назначение |
|--------|------------|
| **Старт** | Стартовая страница приложения. |
| **Реестр ТКП** | Просмотр, поиск и скачивание ранее сформированных ТКП; переход к формированию договора. |
| **Реестр договоров** | Просмотр и скачивание сформированных договоров, черновики. |
| **Формирование ТКП** | Подраздел меню: |
| → ТКП на одну услугу | Создание ТКП по выбранной услуге (типовой шаблон). |
| → Комплексное ТКП | Создание ТКП с произвольной таблицей позиций. |
| → Тарифы по регионам | Настройка цен по услугам и регионам для расчёта стоимости. |
| → Описания услуг | Редактирование текстов описаний услуг (подставляются в комплексное ТКП). |
| **Канбан** | Канбан-доска текущих сделок: колонки этапов, карточки, перетаскивание. |
| **Контрагенты** | Справочник контрагентов (клиентов). |
| **Загрузка реквизитов** | Загрузка карточки контрагента и распознавание реквизитов. |

---

## 1. Формирование ТКП (типовое — одна услуга)

Пошаговый процесс: заполнение формы → проверка данных → формирование и скачивание файлов.

Откройте в меню **Формирование ТКП** → **ТКП на одну услугу**.

### Шаг 1. Заполнение формы

1. Заполните поля:
   - **Дата ТКП** — дата документа.
   - **Услуга** — выберите вид услуги из списка (от него зависит шаблон и единица измерения).
   - **Регион** — выберите регион для расчёта стоимости по тарифу (для внешнего заказчика).
   - **Наименование клиента** — для внешнего заказчика.
   - **Наименование объекта и характеристика помещений** — при необходимости.
   - **Срок разработки** — при необходимости выберите из списка.
   - **Площадь (м²)** или **Количество (шт)** — в зависимости от услуги; используется для расчёта стоимости.
   - **Произвольный текст** — при необходимости.

**Внутренний заказчик:** отметьте «Является внутренним заказчиком?» — появятся поля «Внутренний клиент» и «Стоимость для внутреннего заказчика». Регион и площадь/количество для расчёта не требуются.

2. Нажмите **Сформировать ТКП**.

### Шаг 2. Подтверждение

На экране отображаются введённые данные и рассчитанная стоимость. Проверьте их.

- **Подтвердить и сформировать ТКП** — перейти к формированию и скачиванию.
- **Вернуться к форме** — изменить данные.

### Шаг 3. Скачивание

После формирования доступны кнопки:

- **Скачать PDF** — скачать ТКП в формате PDF.
- **Скачать DOCX** — скачать ТКП в формате Word.

Дополнительно: **Новое ТКП** (создать ещё одно) и **Перечень ТКП** (перейти в реестр ТКП).

## 2. Комплексное ТКП

ТКП с произвольной таблицей позиций (наименование, единица измерения, количество, цена за единицу, стоимость). Итоговая сумма считается автоматически.

Откройте **Формирование ТКП** → **Комплексное ТКП**.

### Шаг 1. Заполнение формы

1. Заполните общие поля:
   - **Дата ТКП**
   - **Наименование клиента**
   - **Срок разработки** (выпадающий список)
   - **Доп. текст 1** и **Доп. текст 2** (по желанию; если пусто — в документ не попадают).
     - **Доп. текст 1** вставляется в документ до таблицы позиций.
     - **Доп. текст 2** — после таблицы.

2. Заполните таблицу позиций:
   - **Наименование услуги** — текст.
   - **Ед. изм.** — м² или шт.
   - **Количество** — число.
   - **Цена за ед.** — цена за единицу в рублях. Стоимость по строке считается автоматически.

3. При необходимости:
   - **Добавить строку** — новая строка в таблице.
   - Кнопка **×** в строке — удалить строку (должна остаться хотя бы одна).

4. Нажмите **Далее — подтверждение**.

### Шаг 2. Подтверждение

Проверьте дату, клиента, срок, доп. тексты и таблицу с итоговой суммой.

- **Подтвердить и сформировать ТКП** — сформировать документ и перейти к скачиванию.
- **Вернуться к форме** — изменить данные.

### Шаг 3. Скачивание

Аналогично типовому ТКП: можно скачать **PDF** и **DOCX**, перейти к **Новому ТКП** или в **Реестр ТКП**.

## 3. Реестр ТКП

Таблица всех сформированных ТКП с фильтрами, ссылками на файлы и действиями.

Откройте **Реестр ТКП** в меню.

### Колонки

- **Дата** — дата ТКП.
- **Номер документа** — уникальный номер (для черновика — ссылка на редактирование).
- **Клиент** — наименование клиента.
- **Услуга** — вид услуги или «Комплексное ТКП».
- **Сумма** — итоговая стоимость (₽).
- **Статус** — черновик или итоговый документ.
- **Файлы** — ссылки для скачивания .pdf и .docx (для итоговых ТКП).
- **Договор** — кнопка «Сформировать Договор» (если доступен шаблон), «Сохранен черновик договора» или «Договор сформирован» с открытием карточки.
- **Удалить** — удаление записи и файлов ТКП (подтверждение в диалоге).

### Фильтры

Над таблицей: **Дата с** / **Дата по**, **Номер**, **Клиент**, **Услуга**, **Сумма от** / **Сумма до**. Нажмите **Применить** для фильтрации, **Сбросить** — снять фильтры.

## 4. Реестр договоров

Просмотр и скачивание договоров, созданных на основе ТКП.

Откройте **Реестр договоров** в меню.

### Фильтры

- **Клиент** — поиск по части названия.
- **Услуга** — выбор из списка или «Все».
- **Статус** — «Сохранен черновик» или «Договор сформирован».

### Колонки

- **Дата**, **Номер документа**, **Клиент**, **Услуга**, **Сумма**.
- **Статус договора** — кнопка открывает карточку договора (черновик или сформированный).
- **Файлы** — ссылки .docx и .pdf (если договор сформирован).

По клику на номер черновика открывается форма редактирования договора.

---

## 5. Тарифы по регионам

Настройка цен за единицу по парам «Услуга — Регион». Тарифы используются при расчёте стоимости в разделе **ТКП на одну услугу** для внешнего заказчика.

Откройте **Формирование ТКП** → **Тарифы по регионам**.

### Добавление тарифа

1. В блоке **Добавить тариф** выберите **Услугу**.
2. Укажите регион: выберите **Регион** из списка или введите **Новый регион** — будет создан новый регион с указанным названием.
3. Введите **Цена за единицу (₽)**.
4. Нажмите **Добавить**.

### Удаление тарифа

В таблице текущих тарифов нажмите **Удалить** в нужной строке.

## 6. Описания услуг

Редактирование текстовых описаний услуг. Эти тексты подставляются в поле «Комментарий» при формировании комплексного ТКП.

Откройте **Формирование ТКП** → **Описания услуг**. Для каждой услуги можно изменить текст описания и сохранить изменения.

---

## 7. Канбан

Канбан-доска для визуализации текущих сделок по этапам.

Откройте **Канбан** в меню.

- **Колонки** — этапы (например: Заявка, ТКП отправлено, Договор и т.п.). Названия колонок можно редактировать; доступно добавление пользовательских колонок.
- **Карточки** — сделки. Карточку можно перетаскивать между колонками; в карточке отображаются основные поля и при необходимости дополнительные (настраиваемые) поля.
- Изменения позиций и полей сохраняются автоматически.

---

## 8. Контрагенты

Справочник контрагентов (клиентов). Используется для выбора клиента при формировании ТКП и договоров, а также для подстановки реквизитов.

Откройте **Контрагенты** в меню. Доступны просмотр списка и поиск контрагентов.

---

## 9. Загрузка реквизитов

Загрузка карточки (документа) контрагента и автоматическое распознавание реквизитов для последующего использования в ТКП и договорах.

Откройте **Загрузка реквизитов** в меню. Загрузите файл (изображение или документ) с реквизитами; система распознает данные и позволяет сохранить их в справочник контрагентов.
