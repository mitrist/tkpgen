# Генератор ТКП

Web-приложение на Django для автоматической генерации технико-коммерческих предложений (ТКП).

**Инструкция для пользователя:** [docs/ИНСТРУКЦИЯ_ПОЛЬЗОВАТЕЛЯ.md](docs/ИНСТРУКЦИЯ_ПОЛЬЗОВАТЕЛЯ.md) — как формировать ТКП, пользоваться перечнем и тарифами.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Шаблоны в папке `templates_docx/`:
   - `Шаблон 1 ДП.docx`, `Шаблон 2 ДКП.docx`, `Шаблон 3 Навигация.docx`, `Шаблон 4 Контент.docx`, `Шаблон 5 Навигация_стенды.docx`

2. Инициализация услуг:
```bash
python manage.py migrate
python manage.py init_services
```

## Запуск

```bash
python manage.py runserver
```

Откройте http://127.0.0.1:8000/

## Шаблоны договоров (переменные для подстановки)

В шаблоны договоров (например, `Шаблон договора_Дизайн_проект.docx`) можно подставлять переменные в формате `{{ имя }}` (docxtpl/Jinja2). Помимо стандартных полей реквизитов в форму «Реквизиты договора» добавлены выпадающие списки, значения которых передаются в шаблон:

| Placeholder в шаблоне | Описание | Варианты в форме |
|----------------------|----------|------------------|
| `{{ customer_in_person }}` | Заказчик в лице | Директора, Генерального директора, И.О. Директора, Подписанта |
| `{{ acting_on_basis }}` | Действующего на основании | Устава, Приказа, Доверенности, ОГРНИП |
| `{{ work_completion_period }}` | Срок выполнения работ | 10 рабочих дней, 20 рабочих дней, 25 рабочих дней, 35 рабочих дней |
| `{{ period_starts_from }}` | Срок отсчитывается от | внесения предоплаты по Договору Заказчиком, с даты согласования макетов Заказчиком |

В тексте шаблона Word укажите, например: «Заказчик в лице {{ customer_in_person }} … действующего на основании {{ acting_on_basis }}».

В шаблонах договоров по комплексному ТКП (**05_Договор_Контент_Навигация.docx**, **08_Договор_ДПФ_Благоустройство.docx**) в нужном месте вставьте абзац с текстом `___CONTRACT_SPEC_TABLE___` — туда будет подставлена таблица спецификации (как в комплексном ТКП).

## Требования

- Windows с установленным MS Word (для конвертации в PDF)
